## 插件说明

### 1. [订阅助手](https://github.com/InfinityPacer/MoviePilot-Plugins/blob/main/plugins.v2/subscribeassistant/README.md)

- 基于[订阅助手](https://github.com/InfinityPacer/MoviePilot-Plugins/blob/main/plugins.v2/subscribeassistant)进行扩展
- 实现多场景管理系统订阅与状态同步
- 本插件仅支持 TMDB 数据源，相关订阅状态说明，请查阅 [#3330](https://github.com/jxxghp/MoviePilot/pull/3330)、[#6015](https://github.com/jxxghp/MoviePilot/pull/6015)
